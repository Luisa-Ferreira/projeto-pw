Seletores CSS

CSS (Cascading Style Sheets) é a linguagem utilizada para estilizar páginas HTML. Permite que alteremos as cores, fontes, layouts e muito mais, proporcionando uma experiência visualmente agradável para todos os que acederem.

Seletor Usado Comentário
Seletor Universal (*) check_circle Estiliza todos os elementos.
Uma das aplicações onde usei foi no 'curso' para atribuir o tipo de letra
Curso
Seletor de Tipo (elemento) check_circle Aplica estilo a todos os elementos de um tipo específico.
Apliquei em todas as aplicações "p" e "h1" etc...
Curso
Seletor de Classe (.class) check_circle Aplica estilo a elementos com uma classe específica.
Apliquei em todas as aplicações
Bandas
Seletor de ID (#id) check_circle Estiliza o elemento com um ID específico
Apliquei mais na aplicação bandas
Bandas
Seletor de Atributo ([atributo]) cancel Estiliza elementos com um atributo específico.
Seletor de Pseudo-classe (:pseudo-classe) cancel Aplica estilo a elementos em um estado específico.
Seletor de Pseudo-elemento (::pseudo-elemento) cancel Estiliza partes específicas de um elemento.

Combinadores de Seletores

Combinador Usado Comentário
Descendente (espaço) cancel Seleciona elementos que são descendentes de um elemento específico.
Filho (>) cancel Seleciona elementos que são filhos diretos de um elemento específico.
Irmão adjacente (+) cancel Seleciona elementos que são irmãos adjacentes de um elemento específico.
Irmão geral (~) cancel Seleciona elementos que são irmãos de um elemento específico, não necessariamente adjacentes.
